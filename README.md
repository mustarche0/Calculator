Java Calculator

This is a simple calculator application developed using Java. It performs basic arithmetic operations. This project was created as part of my learning in Java programming.

Features:
- Addition
- Subtraction
- Multiplication
- Division
- Square Root
- Percentage

Concepts Used:
- Variables and data types
- Conditional statements (if-else)
- User input (Scanner)
- Methods/functions
